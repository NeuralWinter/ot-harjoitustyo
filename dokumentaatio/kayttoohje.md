# Käyttöohje

## Asennus

Kloonaa repositorio ja siirry hakemistoon:
```
git clone https://github.com/NeuralWinter/ot-harjoitustyo.git
cd ot-harjoitustyo
```

Asenna riippuvuudet:
```
poetry install
´´´

## Käynnistys

```
poetry run invoke start
´´´

## Käyttö

### Uuden hahmon luonti

1. Klikkaa **"Create New Character"**
2. Syötä hahmon nimi ja paina **"Next"**
3. Valitse hahmon rotu ja paina **"Next"**
4. Valitse hahmon luokka ja paina **"Next"**
5. Valitse hahmon tausta ja paina **"Next"**
6. Valitse stat-metodi (Standard Array tai nopanheitto) ja jaa arvot
7. Valitse skill proficiencyt luokan sallituista skilleistä
8. Tarkista hahmon yhteenveto ja paina **"Save Character"**

### Tallennetun hahmon lataaminen

1. Klikkaa **"Load Character"**
2. Valitse listasta haluamasi hahmo
3. Hahmon tiedot näkyvät ruudulla

### Takaisin navigointi

Jokaisessa vaiheessa on **"Back"**-nappi, jolla voi palata edelliseen vaiheeseen.
